## 3. 실행 방법 (How to Run)

### 개발 서버 실행
```bash
npm run dev
```

### 새로운 PDF 추가 및 변환
1. `data/raw` 폴더에 PDF 파일 넣기.
2. 변환 스크립트 실행:
```bash
node scripts/convert-pdf.js
```
3. `data/` 폴더에 생성된 `.md` 파일 확인.
